genrndstr

Generate structures on the provided augmented lattice randomly.

-h Display more help (Default: Off)
-laug=[string] Input file defining the augmented lattice (default: lat.in)
-lhs=[string] Input file defining the high-symmetry unstable lattice (default: hs_lat.in)
-ss=[real],... Supercell size for generated structure (3 integer seprated by comma) (Default: )
-sd=[int] Seed for random number generation (default: use clock)
-os=[string] Output file defining the atomic position for the generated structure (default: str.in)
-vac Print out vacancies in the generated structure file (Default: Off)
