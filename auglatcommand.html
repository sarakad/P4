auglat

Make augmented lattice given the high symmetry unstable lattice and a local distortion close by.

-h Display more help (Default: Off)
-l=[string] Input file defining the high-symmetry lattice (default: )
-loc=[string] Input file defining atomic position for the nearby local distortion (default: )
-ol=[string] Output file defining the generated augmented lattice (default: )
